# Political Sentiment Analysis using BERT

This project performs sentiment analysis on political text using transformer models.

Models Used:
- TF-IDF + Logistic Regression
- BERT (bert-base-multilingual-cased)
- DistilBERT
- XLM-RoBERTa

Environment:
Google Colab
Python
HuggingFace Transformers
